# Security and Privacy Guide for Platform Organizers

## Key Responsibilities
- Enforce invitation-only registration by default unless explicitly opened.
- Maintain privacy notices and disclose any trackers or external services used.
- Review access controls and roles; audit elevated permissions regularly.

## Operational Checks
- Run security scans (Brakeman, dependency audits) as part of deployments.
- Ensure backups, retention policies, and deletion workflows meet policy requirements.
- Monitor authentication and notification systems for failures or abuse signals.
